## 한국어 {#korean}
이 저장소는 TliCompendium 앱의 커뮤니티 번역을 포함합니다 (Torchlight Infinite VoidChart & Build Planner 도구).  
또한 이 저장소는 한국어 번역에 대해서만 포크를 게시합니다.
현재 영어(기본), 중국어, 대만어, 러시아어, 한국어, 일본어를 지원합니다.  

파일은 .ts 파일로 저장됩니다 (en.ts, cn.ts, tw.ts, ru.ts, ko.ts, ja.ts).  
기여: 저장소 포크, 게이머에게 자연스럽게 번역, PR 제출.  

**Voidplanner 확인:** [https://tlicompendium.com/build-planner](https://tlicompendium.com/build-planner)  
**Buildplanner 확인:** [https://tlicompendium.com/voidchart-planner](https://tlicompendium.com/voidchart-planner)
